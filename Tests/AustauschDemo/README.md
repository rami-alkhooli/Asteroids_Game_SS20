# Beispiel-Programm zum Datenaustausch zwischen GUI und Datenbank

## Ziel
Einfache Anwendung zum Datenaustausch, um eine Übersicht zu kriegen

## Stand
11.05.2020

## Bedienung
- Mit 1 erhöht man lokal das Highscore
- Mit 2 erhöht man lokal das Score
- Mit 3 erhöht man lokal die Anzahl der Shots
- Mit 4 erhöht man lokal die Anzahl der Items
- Mit 5 erhöht man lokal die gespielte Zeit

- Mit s sollte man die lokalen Daten an die Datenbank schicken können
- Mit r sollte man die in der Datenbank gespeicherten Daten empfangen und in die lokalen Daten laden
